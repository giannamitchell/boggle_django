**Django Backend Solver**
Hello, my name is Gianna and here is my project!
This is a backend web service built using the Django framework.

*Note on comments: I orginally worked on this project through Codio and pushed it to GitHub which is why commits are sparse*

How it works: The backend accepts a 4x4 matrix representation Boggle board and using tests built around the boggle game logic returns all valid words.

*also includes Spanish as a language option.

  
### Input Board Layout:
```text
M  S  E  T
R  A  T  G
L  O  I  E
F  Q  N  W
```

### Backend Output Response:
{
  "words_found": ["STAR", "TEAR", "ROAST", "MARS", "TIGER", "RATS"],
  "total_count": 6
}
